## 冒泡算法
冒泡算法的思路是从首元素开始，依次讲当前元素与后一个元素相比较，前者大于后者则交换位置。  
共遍历n次即可。  
```
 for(int i = 0;i < n;i ++)
    {
        for(int j = 0;j< n - i - 1;j ++)
        {
            if(array[j] > array[j + 1])
            {
                int temp = array[j];
                array[j] = array[j + 1];
                array[j + 1] = temp;
            }
        }
    }
```
